- install [ripgrep](https://github.com/BurntSushi/ripgrep),
- install missing software mentioned in `:checkhealth`.
- edit lua/phanxor/links.lua to fix reference errors.

- colorscheme stuff
- automatic symbol pairing
- neo-tree sidebar
- barbar for tabs (clickable)
- which-key
- zenmode for editing without distractions
- noice for improving uis
- telescope for finding files
- trouble for errors
